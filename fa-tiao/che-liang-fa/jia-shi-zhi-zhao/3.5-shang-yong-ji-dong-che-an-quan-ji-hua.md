# 3.5 商用机动车安全计划

### 1. 根据本条法律，雇主不得故意允许、要求或授权驾驶员在下列情况下驾驶商用机动车：

  1. 驾驶员的驾驶执照或特权被任何州暂停、吊销或取消，或已被取消驾驶商用机动车的资格；
  2. 驾驶员持有超过一张驾驶执照；


### 2. 根据本条法律，任何人不得驾驶商用机动车辆，除非该人直接拥有相应类别的有效商业驾驶执照；


### 3. 根据本条法律，任何人不得在运输危险材料时驾驶商用机动车辆，除非该人持有带有危险材料背书的有效商业驾驶执照。指令许可证不授权运输危险材料的车辆运行；


### 4. 根据本条法律，执法人员有权在任何情况下拦停一台商用机动车以进行安全检查；


### 5. 根据本条法律，商用机动车驾驶员应随车携带以下所有文件：

  1. 车辆注册文件；
  2. 车辆保险证明；
  3. 货运清单（如果商用机动车为货运性质）；
  4. 详细记录行驶路线、行驶时间（不得连续超过四个小时）、休息时间（不得少于四十分钟）的工作日志（如果商用机动车为货运性质）。

***

{% hint style="warning" %}
违反车辆法第3.5条的行为属违法，应判处最高$5,000罚款并扣押其车辆。
{% endhint %}

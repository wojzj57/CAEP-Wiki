# 2.2 政府机构雇员受贿罪

### 1. 据本条法律，本州任何政府机构雇员要求、接受贿赂的行为，构成犯罪；


### 2. 据本条法律，本州任何政府机构雇员同意接受任何贿赂的行为，构成犯罪；

***

{% hint style="danger" %}
若行政人员实际接受了贿赂，属重罪，应判处最高四年监禁和（或）至少$2,000罚款；
{% endhint %}

{% hint style="danger" %}
若行政人员没有实际接受贿赂，属轻罪，应判处最高一年监禁和（或）$10,000罚款。
{% endhint %}
